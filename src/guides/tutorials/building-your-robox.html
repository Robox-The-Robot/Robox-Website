Components

Here are a list of components required to build your own Robox:

  • 1x L298n Motor Driver
  • 2x TT DC Motor
  • 1x RCWL-1601 Ultrasonic Sensor
  • 1x 3AAA Battery Holder
  • 2x IR Reflectance Sensor
  • 1x Raspberry Pi Pico
  • A set of Male-Female, Female-Female and Male-Male wires

Additionally, you will also need some other items around the house:

  • 1x Small flathead screwdriver
  • 5x Plastic bottle caps
  • 2x Rubber bands
  • 1x Toothpick
  • 1x Roll of tape (preferably double sided)

You can find this component list, and Robox cutouts here.

How does Robox work?

Robox is a line-following robot, which means it follows a black line on the ground. Since it has to move, we need motors which can spin the wheels!

The motor has red and black wires, which are used for electricity to flow through. When electricity goes through the motors, it will start spinning.

But how do we control the motors? We need to use a motor driver, which can deliver the electricity to the motor. We have to send a pulse to the motor driver from a microcontroller (small robotics computer) to the driver to tell it to start moving the motors.
