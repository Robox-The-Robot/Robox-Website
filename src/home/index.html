What is Robox?

Robox is an ultra-cheap, easy to use line-following robotics kit that can be used to teach STEM to anyone. Most robotics kits are expensive and out of budget for many schools and students, and Robox aims to break cost barriers. By reusing components from around the house and using cheap components, one Robox kit can be built for under US$8. Additionally, Robox is designed to be easy to build and use, with no soldering required and a simple design.

Robox is designed to be used with the Raspberry Pi Pico, a cheap microcontroller that can be programmed in C or MicroPython. Part of the Robox project is to provide free resources and guides to help you get started with robotics and programming!
